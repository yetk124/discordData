# discordData

# Discord Sudden Attack Bot

> 🎯 **디스코드에서 Sudden Attack 전적을 조회하고, 채팅을 관리할 수 있는 봇**

## 📌 주요 기능
1. **🗑️ 메시지 삭제 기능**
   - 특정 사용자가 채널에서 지정한 개수만큼 메시지를 삭제 가능 (`/청소 <개수>`)

2. **🔎 Sudden Attack 전적 조회**
   - 닉네임 또는 ID를 입력하면, 해당 플레이어의 **K/D 정보, 모드별 성적** 등을 조회 (`/병영수첩 <유저 ID>`)
